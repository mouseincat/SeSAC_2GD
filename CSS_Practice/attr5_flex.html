  1. CSS3 부터 나타난 방식
  2. 화면 레이아웃을 구성할 때 가장 많이 사용하는 방법
  3. 부모요소에(container) flex를 선언하면 자식 요소의 레이아웃이 변경됨

flex-container

  • flex속성을 갖는 요소, flex 부모요소
  • 컨테이너에 적용 가능한 속성이 있음
    1. flex-direction
    2. flex-wrap
    3. justify-content
    4. align-items
    5. align-content (flex-wrap과 같이 써야함)

flex-item

  • flex 속성을 가지는 container의 자식 요소들
  • 아이템에 적용 가능한 속성이 있음.
    1. order
    2. align-self
    3. flex-basis
